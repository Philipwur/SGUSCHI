This directory shows a typical starting point for running SGUSCHI simulations.

To start from this point, do the following:

1. Run PrepareWorkplace.py in this folder (in preprocessing folder)
2. Manually submit a job in each Dir_VolSearch (e.g. sbatch jobsub)
3. Run OxidationMaster (e.g. sbatch OxidationMaster)
4. Results will be output in xyz_files
5. Resubmit OxidationMaster until simulations are of a sufficient length
